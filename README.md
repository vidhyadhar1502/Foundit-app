# 🔍 FoundIt

A web-based application to help users **find and match items** — built with [Base44](https://base44.app) and deployed as a standalone HTML app.

---

## 🌐 Live Demo

👉 [https://find-match-found.base44.app](https://find-match-found.base44.app)

Or open `FoundIt.html` directly in your browser after cloning this repository.

---

## 📋 About

**FoundIt** is a lightweight front-end web app designed to help users find matches or locate items quickly. It runs entirely in the browser with no backend required.

---

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No installations or dependencies needed

### Run Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/YOUR_USERNAME/foundit.git
   cd foundit
   ```

2. **Open the app:**

   Simply double-click `FoundIt.html` or open it in your browser:

   ```bash
   open FoundIt.html        # macOS
   start FoundIt.html       # Windows
   xdg-open FoundIt.html    # Linux
   ```

## 🌍 Deploy with GitHub Pages

Host this app for free using GitHub Pages:

1. Push your code to a GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Branch**, select `main` and click **Save**.
4. Your app will be live at:

   ```
   https://YOUR_USERNAME.github.io/foundit/FoundIt.html
   ```
